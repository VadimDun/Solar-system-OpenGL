# Solar-system-OpenGL
Лабораторная работа №13 по компьютерной графике: Загрузка файлов, множественные объекты

## Задания
Для выполнения [задания](https://edu.mmcs.sfedu.ru/mod/assign/view.php?id=29761) надо сделать следующее:
- Загрузка моделей из файлов
- Отрисовка множественных моделей
- Движение по орбите
- Камера

## Стек
* **Язык программирования**: C++17
* **Графический API**: OpenGL 3.3+
* **Оконная система**: SFML 2.6.1
* **Расширения**: GLEW для загрузки OpenGL функций
* **Шейдеры**: GLSL 330 core

## Первая сборка проекта:
``` bash
cd build
cmake ..    # ← Генерация (один раз)
make        # ← Компиляция
```

## После изменения файлов в **/3d-objects**:
```bash
make        # ← Пересборка (cmake уже не нужен)
```
